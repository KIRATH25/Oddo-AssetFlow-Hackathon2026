# 🚀 AssetFlow - Enterprise Asset & Resource Management System

AssetFlow is a centralized Enterprise Resource Planning (ERP) platform designed to simplify asset and resource management for organizations. It enables businesses to efficiently track assets, manage allocations, schedule shared resources, handle maintenance workflows, and conduct audit cycles.

---

## 📌 Problem Statement

Organizations often rely on spreadsheets and manual records to manage assets, resulting in inefficient tracking, conflicts, and poor visibility.

AssetFlow provides a centralized solution to:

- Track asset lifecycle
- Allocate assets to employees/departments
- Manage shared resource bookings
- Handle maintenance approvals
- Conduct audit cycles
- Generate operational reports
- Maintain activity logs and notifications

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/AssetFlow.git
```

## Frontend

```bash
cd Frontend
npm install
ng serve
```

## Backend

```bash
cd Backend
dotnet restore
dotnet run
```

## Database

- Open SQL Server Management Studio
- Create Database

```sql
CREATE DATABASE AssetFlowDB;
```

- Execute all SQL scripts.
